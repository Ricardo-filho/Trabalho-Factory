# Trabalho-Factory

Link canvas: https://canva.link/rgpi3pczeggavhe

O Factory Method é um padrão de projeto que centraliza a criação de objetos em uma classe específica, chamada fábrica. Em vez de instanciar objetos diretamente com new, o sistema solicita que a fábrica crie o objeto desejado. Isso torna o código mais organizado, reduz o acoplamento e facilita a manutenção e a adição de novos tipos sem alterar a lógica princi
